# 智源Talk 内容生成器 (Zhiyuan Talk Generator)

> WorkBuddy Skill · v1.2.0

为智源Talk活动自动生成内容介绍文案及标准格式 md 文件。

## 功能

- **内容介绍文案生成**：根据研究素材自动生成 ≤230 字的精炼宣传文案
- **md 文件生成**：一键输出包含报告信息、嘉宾简介、内容介绍、相关论文、工作摘要的完整 md 文件
- **吸睛标题建议**：可选生成营销风格的公众号标题（≤45字）

## 安装

1. 下载本仓库中的 `智源Talk内容生成器.zip`
2. 打开 WorkBuddy，进入 Skill 管理
3. 选择「从文件导入」，选择下载的 zip 文件
4. 安装完成后即可使用

## 使用

在 WorkBuddy 中输入以下触发词即可启动：

```
智源Talk内容生成、智源Talk文案、智源talk介绍、活动介绍文案、
智源Talk预告、智源Talk邀请词、智源talk宣传、智源Talk、
生成活动md、智源Talk md文件
```

## 工作流程

```
阶段一：生成内容介绍文案（≤230字）
  → 用户审核确认
阶段二（可选）：生成吸睛标题（≤45字）
  → 用户选择或跳过
阶段三：生成完整 md 文件
```
